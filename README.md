# Game Store Project
A Flask-based web application for an online game store.

## Features
- User authentication
- Game search by title
- Game purchase and library
- Admin and developer dashboards

## Setup
1. Clone: `git clone https://github.com/ordabekuly/BAC-END-END-TERM.git`
2. Install: `pip install -r requirements.txt`
3. Run: `python run.py`
